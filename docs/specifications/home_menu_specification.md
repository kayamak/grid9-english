# ホーム画面コマンドメニュー仕様

## 1. メニュー構成

コマンドメニューは以下の2階層構成とする。

| 第1階層 (カテゴリー) | 第2階層 (アクション) | 遷移先 |
| :--- | :--- | :--- |
| たたかう | ドリルクエスト | `/practice?mode=quest` |
| じゅもん | DO_SV | `/practice?mode=drill&pattern=DO_SV` |
| | DO_SVO | `/practice?mode=drill&pattern=DO_SVO` |
| | BE_SVC | `/practice?mode=drill&pattern=BE_SVC` |
| どうぐ | じゆうトレーニング | `/practice?mode=free` |

## 2. インタラクション仕様

### 2.1 コマンドメニューの表示
- 初期表示時は「第1階層」のみを表示する。
- 第1階層の項目をクリックすることで、その配下の「第2階層」を表示する。
- 第2階層の項目をクリックした際に、各遷移先へ画面遷移を行う。

### 2.2 右側表示領域（説明エリア）
- 第1階層または第2階層の項目が選択・クリックされる度に、その内容に応じた説明文を右側の表示領域に表示する。
- **追記要件**: 「説明をクリックする度に切り替える」という挙動については、選択されたコマンドの詳細説明を順次、またはランダムに切り替えるものとする。

### 2.3 下部メッセージウィンドウ
- 初期表示メッセージ: 「ぼうけんの　じゅんびは　いいかな？」
- コマンドメニューがクリックされる度に、モチベーションを上げる言葉にランダムに切り替える。
- モチベーションを上げる言葉の例:
  - 「きみなら　できる！」
  - 「あきらめたら　そこで　しあいしゅうりょうだよ」
  - 「一歩ずつの　積み重ねが　大きな力になる！」
  - 「伝説の　勇者は　君のことだ！」
  - 「英語の　呪文を　マスターしよう！」

## 3. デザイン（DQ風スタイル）
- 背景: 黒 (#000000 または 濃紺 #000840)
- 枠線: 二重枠、白
- フォント: DotGothic16 (ドット文字)
- カーソル: 「▶」などのアイコンを使用
