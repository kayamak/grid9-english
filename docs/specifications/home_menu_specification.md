# ホーム画面コマンドメニュー仕様

## 1. メニュー構成

コマンドメニューは以下の2階層構成とする。

| 第1階層 (カテゴリー) | 第2階層 (アクション) | 遷移先 |
| :--- | :--- | :--- |
| たたかう | ドリルクエスト | `/practice?mode=quest` |
| れんしゅう | DO_SV | `/practice?mode=drill&pattern=DO_SV` |
| | DO_SVO | `/practice?mode=drill&pattern=DO_SVO` |
| | BE_SVC | `/practice?mode=drill&pattern=BE_SVC` |
| おためし | じゆうトレーニング | `/practice?mode=free` |
| (その他) | にげる (レベルクリア) | (モーダル表示) |

## 2. インタラクション仕様

### 2.1 コマンドメニューの表示
- 初期表示時は「第1階層」のみを表示する。
- 第1階層の項目をクリックすることで、その配下の「第2階層」を表示する。
- 第2階層の項目をクリックした際に、各遷移先へ画面遷移を行う。

### 2.2 右側表示領域（説明エリア）
- 第1階層または第2階層の項目が選択される度に、その内容に応じた説明文を右側の表示領域に表示する。
- 説明表示領域をクリックすると、操作方法に関するメッセージに切り替わる。

### 2.3 レベルクリア（にげる）
- トップメニューに表示される「にげる」を選択すると、確認用のモーダルが表示される。
- モーダル内の「レベルクリア」を実行すると、プレイヤーレベルが1にリセットされる（確認ダイアログなしで即時実行）。

### 2.4 下部メッセージウィンドウ
- 初期表示メッセージ: 「ぼうけんの　じゅんびは　いいかな？」
- コマンドメニューをクリックする度に、モチベーションを上げる言葉に切り替える。
- メッセージウィンドウをクリックすると、操作方法に関するメッセージに切り替わる。
- モチベーションを上げる言葉の例:
  - 「きみなら　できる！」
  - 「あきらめたら　そこで　しあいしゅうりょうだよ」
  - 「一歩ずつの　積み重ねが　大きな力になる！」
  - 「伝説の　勇者は　君のことだ！」

## 3. デザイン（DQ風スタイル）
- 背景: 黒 (#000000 または 濃紺 #000840)
- 枠線: 二重枠、白
- フォント: DotGothic16 (ドット文字)
- カーソル: 「▶」などのアイコンを使用
